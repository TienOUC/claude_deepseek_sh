#  1. Mac 执行步骤
1. 打开 终端
2. 输入 `chmod +x mac_install.sh` 回车
3. 输入 `./mac_install.sh` 回车
4. 执行过程中根据提示输入 `DeepSeek API Key（sk-开头）`

# 2. Windows 执行步骤
1. 以管理员身份打开 PowerShell
2. 输入 `.\windows_install.ps1` 回车
3. 执行过程中根据提示输入 `DeepSeek API Key（sk-开头）`

# 3. 获取 DeepSeek API Key
1. 登录：https://platform.deepseek.com 注册账号
2. 进入「API Keys」→ 新建密钥
3. 复制 sk-xxxxxx 格式密钥，保存好
